Aula 02
Colocando Imagem no fundo da Página

	Usando a propriedade background-image com a 
função url, podemos adicionar uma imagem no fundo
do site.
CSS
	div {
		background-image: url('imagens/imagem.jpg');
	}

Aula 03
Colocando Imagem com uma URL

body {
    background-image: url('https://gustavoguanabara.
	github.io/html-css/imagens/mascote.png');
	background-size: 100px 112px;
}

	A repetição das imagens é Padrão, a não ser que
mude para background-repeat: no-repeat;. Também
da para repetir horizontalmente ou verticalmente,
usando os valores "repeat-x" ou "repeat-y".


Aula 04									28/05/25

Background-Position

	Aprendendo a posicionar os fundos, a ordem dos
valores é horizontal depois vertical, os valores
padrão são "left top".

ex:
div.bloco {
	background-image: url('imagens/imagem.jpg');
	background-position: left bottom;
}

	os valores horizontais: left center right
	os valores verticais: top center bottom
	
	Importar lembrar que quando for usar imagens
no body e quiser que ela preencha toda a página
colocar "height: 97vh;", pois 100vh cria uma rolagem
na página. vh = view height.

Aula 05

	É Importante adicionar uma cor no fundo, mesmo
que adicione uma imagem também, pois caso o seu
servidor esteja lento o usuário poderá acessar o
conteúdo enquanto carrega as imagens.

	background-size com o valor "cover", adapta a
imagem de acordo com o tamanho da tela.

Valores:
 - auto: Usa o tamanho original da imagem (padrão).
 - cover: A imagem cobre toda a área do elemento.
		Pode cortar a imagem.
 - contain: A imagem é redimensionada para caber
		inteira dentro do elemento.
 - largura altura: Define manualmente o tamanho.
		Ex: 100px 50px ou 50% auto.
		
Shothand Background com os valores na sequência:

background-image, background-repeat,
background-position, background-attachment.
