Aula 02
Colocando Imagem no fundo da Página

	Usando a propriedade background-image com a 
função url, podemos adicionar uma imagem no fundo
do site.
CSS
	div {
		background-image: url('imagens/imagem.jpg');
	}

Aula 04									28/05/25

Background-Position

	Aprendendo a posicionar os fundos, a ordem dos
valores é horizontal depois vertical, os valores
padrão são "left top".

ex:
div.bloco {
	background-image: url('imagens/imagem.jpg');
	background-position: left bottom;
}

	os valores horizontais: left center right
	os valores verticais: top center bottom
	
	Importar lembrar que quando for usar imagens
no body e quiser que ela preencha toda a página
colocar "height: 100vh;", porém é necessário usar
o asterisco para criar um seletor universal com 
"box-sizing: border-box;!
para evitar rolagem na página. vh = view height.

Aula 05

	É Importante adicionar uma cor no fundo, mesmo
que adicione uma imagem também, pois caso o seu
servidor esteja lento o usuário poderá acessar o
conteúdo enquanto carrega as imagens.

	background-size com o valor "cover", adapta a
imagem de acordo com o tamanho da tela.

Valores:
 - auto: Usa o tamanho original da imagem (padrão).
 - cover: A imagem cobre toda a área do elemento.
		Pode cortar a imagem.
 - contain: A imagem é redimensionada para caber
		inteira dentro do elemento, sem cortes.
 - largura altura: Define manualmente o tamanho.
		Ex: 100px 50px ou 50% auto.

Aula 06

background-attachment

	Determina se o fundo vai rolar junto com o
conteúdo da página ou se vai ficar fixo no lugar,
como se fosse um papel de parede.
	
	Valores:
- scroll(Padrão): A imagem de fundo rola junto com
		o conteúdo da página.
- fixed: A imagem de fundo fica fixa na tela, mesmo
		rolando a página.
- local: O fundo rola junto com o conteúdo da área
		de rolagem do elemento.
		
	A combinação do attachment "fixed" com o
background-size "cover" fica com um resultado muito
bom.

Shothand Background com os valores na sequência:

color > image > repeat > position > attachment.

Aula 07

Centralizando divs

	Mesma forma que foi feita em "Vídeos Centralizados"
no dia 21/05/25. Porém apenas usando <div> sem o <iframe>.

HTML
<div class="box">
	<div>
	
CSS
* {
	margin: 0;
	padding: 0;
	box-sizing: border-box;
}

.box {
	position: relative;
	max-width: 800px;
	min-width: 260px;
	background-color: cor-escolhida;
	margin: auto;
}
.box div {
	position: absolute;
	max-width: 400px;
	min-width: 130px;
	top: 50%;
	left: 50%;
	transform: translate(-50%, -50%);
}
